# 🧮 Calculadora React
Projeto desenvolvido como parte do curso de Formação React Developer da Digital Innovation One-DIO, com implementação própria.

## 🚀 Funcionalidades

- Operações básicas: +, -, *, /
- Suporte a números negativos
- Validação para múltiplos operadores
- Bloqueio de múltiplos pontos decimais
- Suporte a teclado (incluindo Enter)
- Botão de limpar (CC)

## 🛠️ Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS Grid

## 📂 Estrutura

- `App.jsx` → lógica principal
- `App.css` → estilização da calculadora

## ▶️ Como executar o projeto

```bash
npm install
npm run dev
